## Buat Project Laravel
### 1. Masuk ke folder
```bash
cd C:\laragon\www
```
### 2. Buat project
```bash
composer create-project laravel/laravel cuti-frontend
```
### 3. Jalanin Project
```bash
php artisan serve
```
### 4. Buat Fitur PDF
```bash
composer require barryvdh/laravel-dompdf
```
